### What I learned

I learnt more on how to use CSS flexbox and grid to build responsive layouts. I also learnt some new CSS animation properties
```css
.class-name {
    display: grid;
    gap: 20px;
    grid-template-columns: repeat(2, 1fr); 
}
```
